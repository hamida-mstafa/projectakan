The Akan Naming System

This is a website to determine your name based on the day of the wek you were born. This ghanian system is called the akan naming system
If you are male you would be named as follows:

Sunday-Kwasi, Monday-Kwadwo, Tuesday-Kwabena, Wednesday-Kwaku, Thursday-Yaw, Friday-Kofi, saturday-Kwame

Else if you are female:

Sunday-Akosua, Monday-Adwoa, Tuesday-Abenaa, Wednesday-Akua, Thursday-Yaa, Friday-Afua, Saturday-Ama
